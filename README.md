## Overview
Hexagoanl Chess engine written in C.

## Features
- No opening book
- Single Piece-Square Table Evaluation
- Configurable alpha-beta search depth
- Stale game evaluation
- No Castling in Hex Chess 
- En passant
- ASCII display
